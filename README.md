## Installation

Application requires [Node.js](https://nodejs.org/) v10+ and [Git](https://git-scm.com/) to run 

Install the dependencies and devDependencies and start the server.

```sh
git clone https://github.com/kazimahin/mui-dashboard
cd mui-dashboard
npm i 
npm start
```

### All dependencys are 

- @material-ui/core
- @material-ui/icons
- react -router-dom
- font awsome cdn






## Features
